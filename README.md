Tela de Login/Cadastro funcional e com validação utilizando conceitos de POO.
Funcionalidades:
- Salvar novo usuário no banco de dados;
- Filtrar se usuário esta cadastrado e fazer login;
- Validar e verificar se todos os campos estão preenchidos.

Requisitos:
- Java IDE (Utilizei Apache Netbeans 28)
- MySQL
- MySQL Connector
